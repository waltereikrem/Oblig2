CSS Layouts

Det finnes flere måter å bruke layout i HTML. I HTML bruker man et språk som heter CSS til å stilere siden og strukturere siden. alt fra farger til bredde og possisjoner styres av css.

I dag det flere metoder som blir brukt. f.eks; Grid, Flexbox eller float. Float er et gammelt system som ikke egentlig var ment til å brukes til layout. men siden det fungerte ble det tatt i bruk, slik at ting ble mye enklere. Med float så sier man via CSS hvilken side elementet skal klistres til. feks "float: left;" eller lignende. man kan også si at elementet skal være uten andre elementer på siden av seg. Problemet med float er hvor tungvindt det er å få ting nøyaktig og pent. derfor fikk vi CSS Grid og Flexbox.

CSS Grid og Flexbox er systemer designet for å gjøre ting raskere, penere og bedre. Noe det definnitivt gjør. disse metodene lager et strukturert områte som reagerer på hverandre og foreldre elementene. noe som gir mye større muligheter for responsivt design. Det var såklart mulig med float også, men det ville ta uandelig mye mer tid. med disse systemene lager man netverk som sorterer og lar deg plotte inn hvor du vill ha elementene dine kunn ved å skrive hvor de skal. Noe som ville tatt vanvittig med kode for å få til like bra om man brukte float systemet.
