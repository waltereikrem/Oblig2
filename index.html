Git / GitHub

Teknologien Git og spesielt tjenester som Github, Bitbucket, Gitlab og lignenende er svært viktig i utviklingsverden. Teknologien Git i sin enkleste form er en måte å holde styr på hva man gjør i et prosjekt, ved å legge filene sine i det som kalles et Git repository så får man mulighet til å se endringene man har gjort i prosjektet over tid, dette kalles versjonskontroll. Tjenester som Github lar deg gjøre dette på tvers av et utviklingsteam som gjør at alle kan se hvilke endringer som er gjort i prosjektet og hvem som har gjort disse endringene.

CSS Layouts

I dag det flere metoder som blir brukt. f.eks; Grid, Flexbox eller float. Float er et gammelt system som ikke egentlig var ment til å brukes til layout. men siden det fungerte ble det tatt i bruk, slik at ting ble mye enklere. Med float så sier man via CSS hvilken side elementet skal klistres til. feks "float: left;" eller lignende. man kan også si at elementet skal være uten andre elementer på siden av seg. Problemet med float er hvor tungvindt det er å få ting nøyaktig og pent. derfor fikk vi CSS Grid og Flexbox.

Media Typer

Før media queries var det veldig vanlig å lage separate css stilark for alle de forskjellige enhetene, vi fikk derfor media typer som screen (for skjermer), handheld (for mobil) men og en hel del andre som for skjermlesere, braille for blindeskrift, og andre for spesielle bruksområder.
