Git

for å forstå hva GitHub er, må du først sette deg inn i og få en forståelse for hva Git er. Git er et form for versons kontroll system laget av samme mannen som lagde Linux. Nemlig Linus Torvalds. Git er veldig likt mye annet der ute. så det finnes mange varianter av versonskontroll.

Versjonskontroll systemer holder styr på koden for en uvikler. Git holder styr på at ikke en redigert versjon blir ut av det blå slått sammen eller skaper konflink med en annen redigert versjon av samme fil. Dette er supernyttig for noen som jobber i store kode teams. Da kan en eller flere kode på samme proskjekt og likevell ikke ødelegge for hverandre, samt alle involvert vill kunne se hhva som skjer og sammeligne sin verskjon og holde seg oppdatert.

GitHub

nå som du vet hva Git er. Lurer du kanskje på hva GitHub er og hva som gjør det så spesielt. Git er bare selve oppgaven og tingen som gjøres. GitHub blir da verktøyet man bruker for å gjøre git mer enkelt og brukervenlig. og GitHub er sentrert rundt GitHub.com - et sted å lagre koden og kommunisere med andre utviklere.
